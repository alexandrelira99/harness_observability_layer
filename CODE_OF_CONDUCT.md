# Code Of Conduct

This project aims to be collaborative, respectful, and technically honest.

## Our Standards

- be respectful and constructive
- focus criticism on ideas, code, and behavior, not people
- assume good intent while still being clear about risks and tradeoffs
- welcome questions from contributors at any experience level

## Unacceptable Behavior

- harassment, threats, or discriminatory behavior
- personal attacks or sustained hostility
- publishing private or sensitive information without permission
- bad-faith review behavior intended to block or belittle contributors

## Enforcement

Project maintainers may moderate discussions, request changes in conduct, or remove contributions that violate these expectations.

For serious concerns, contact the maintainer privately.
